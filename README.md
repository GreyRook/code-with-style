# Python Coding Guide


 * Prefer Python Version: 3.6
 * Prefer `pipenv + Pipefile` over `pip + requirements.txt`
 * We follow [PEP 8](https://www.python.org/dev/peps/pep-0008/).


 ## Preferred libraries


Library | Use Case
--------|--------------------
Pillow  | Image manipulation
pytest  | unit text execution
