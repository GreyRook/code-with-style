# Setup

```sh
mkdir -p ~/.cookiecutters

git clone --single-branch --branch master https://github.com/FlorianLudwig/cookiecutter-py3.git ~/.cookiecutters/py3
git clone --single-branch --branch aiohttp https://github.com/FlorianLudwig/cookiecutter-py3.git ~/.cookiecutters/py3-aiohttp
```

# Usage

```sh
cookiecutter py3
cookiecutter py3-aiohttp
```
